项目仓库

一、环境准备

1. idea

  1. 设置

    • view中勾选tool buttons和boolbar

    • 如何删除模块？选中模块–>键盘上delete,会把模块从工程中移除，但还在硬盘上–>delete,从硬盘上删除

    • 代码自动导入：settings–>editor–>general–>auto import–>insert imports on paste中选all,勾选classes,staticxxxx,addxxxx,optimizexxx等

    • 忽略大小写提示:settings—>搜索code completion–>在case sensitive completion上选择none

    • 字体大小：settings–>搜索font–>修改大小

    • 工程编码：settings–>搜索file encodings–>把所有编码改为utf-8，勾选transparentxxx

    • 自动编译：settings–>搜索compiler–>勾选build project automatically和compile independent modules in parallel

    • 使用eclipse中常用快捷键 file–>import settings–>找到jar包据在位置

    • 隐藏不想看到的文件 file—>settings—>editor–>file types下的ingore files and folders中添加不想看到的文件

2. maven

ideal设置maven

file–>Build,–>Build Tools –>maven,给右侧三个地方配置成本地值

3. git

  1. 安装

    要安装到一个没有空格且非中文目录下，并选择Use Git from Git Bash only

  2. 设置签名

    • git config –global user.name lly—配置姓名
    • git config –global user.email 123@qq.com—配置邮箱

  3. idea使用git

    • settings –>version control–>git–>在path一栏中找到git.exe位置
    • 添加gitee帐号
      • settings—>Version Control–>Gitee
      • 右侧点“+”号–>Log in via Gitee
      • 填写gitee注册时的邮箱和密码

  4. 项目协作

    • 分享项目到gitee

      1. git–>share project on gitee

      2. 提交到远程库

        git–>push

    • 从gitee克隆项目

      1. vcs–>get from version control–>gitee
      2. URL中，填写远程仓库地址，Directory中填写代码地址。
      3. 点“Clone”开始下载代码

    • 更新本地项目项目

      Git–>Update Project

    • 提交本地项目

      Git–>commit–>commit and push

4. mysql

参考安装包中的安装说明安装。

二、项目功能描述

1、认证业务

  1. 用户注册
  2. 登录

2.购物车

1）浏览商品

用户在网购站点浏览商品信息

2）添加购物车

选择合适的商品,加购物车

3）redis

redis实现购物车

3.生成订单

1）物流地址

获取国内各省市区地址列表，选择物流地址

扩展：新增保存用户常用地址表,关联用户表,一个用户对应多个地址,用户可选择常用地址

2）订单

选取收货地址,生成订单，无需支付,清空购物车

3. 帐单

1）生成帐单

  1. 产生订单后，生成账单,及分期还款信息.
  2. 按分期数和年化30%利息生成多条预还款记录.
  3. 生成账单时扣除相应预授额度,如果预授额度不够,则购买失败,回滚所有前置操作

2）查询账单

查询历史分期还款记录,还款进度.查询7天待还款.

4. 信用额

登录后才可访问

  1. 用户查询自己的预授信额度,已用额度,可用额度，初次访问帐单页面,可看到默认额度为0,
  2. 点击申请提升额度，不可无限提额.

三、项目搭建

1.导入依赖

2. 添加前端页面

  1. 功能页面

    在resources目录中创建子目录templates,在templates目录中创建两个子目录consumer和credit，在两个子目录中放入相应的html页面

  2. 添加登录页面

    在resources目录中，创建static子目录，在static中添加login.html

  3. 添加静态资源

    在static目录中创建img和china两个子目录，并添加对应资源
