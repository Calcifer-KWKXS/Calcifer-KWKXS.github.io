前言

网上整理好的Markdown语法相关资料有很多，在此自己整理记录一遍，以熟悉语法规则。

标题

在想要设置为标题的文字前面加 # 来表示（标准语法一般在# 后跟个空格再写文字）

一个 # 是一级标题，二个 # 是二级标题，以此类推，支持六级标题。

换行

方法2：使用html语言换行标签 <br>

代码

语法：

  1. 单行代码：代码之间分别用一个反引号（`）包起来

  2. 代码块：

列表

语法：

1.无序列表：用 - + 、* 任何一种都可以（符号跟内容之间都要有一个空格）
