前言

网上整理好的Markdown语法相关资料有很多，在此自己整理记录一遍，以熟悉语法规则。

标题

在想要设置为标题的文字前面加 # 来表示（标准语法一般在# 后跟个空格再写文字）

一个 # 是一级标题，二个 # 是二级标题，以此类推，支持六级标题。

换行

方法1: 连续两个以上空格+回车

方法2：使用html语言换行标签 <br>

(我实际体会中两个回车也可以,更喜欢这种)

代码

语法：

  1. 单行代码：代码之间分别用一个反引号（`）包起来

  2. 代码块：

    方法1：代码之间分别用三个反引号包起来，且两边的反引号单独占一行。（可以在前三个反引号后添加语言名，代码会高亮显示）

    方法2：使用 4 个空格或者一个制表符（Tab 键）。

语法：

1.无序列表：用 - + 、* 任何一种都可以（符号跟内容之间都要有一个空格）

插入图片

语法：

![图片描述](图片链接或本地路径 ‘’图片title’’)

图片描述：显示在图片下面的文字，相当于对图片内容的解释。
图片title：图片的标题，当鼠标移到图片上时显示的内容。
注：图片描述 和 title 只是可选项，可以不写

字体

  1. 加粗：**加粗**，效果：加粗

  2. 斜体：*斜体*，效果：斜体

  3. 斜体加粗：***斜粗内容***，效果：斜粗

  4. 删除线：~~删除~~，效果：删除

分割线

三个或者三个以上的 - 或者 * 或者 _ 都可以，显示效果都是一样的。
